Marlon Velasco Resume Website

Type: Personal/ Employment — Website UI Design

Year: 2025

Role/Contributions: UI/UX Designer / Front-end Designer

Tool used: Figma

General Description

The Resume Website is a project that is self-designed personal resume website that presents professional information in an interactive and visually appealing layout. It serves as a digital resume and an online portfolio in one platform. It contain different important information such as personal introduction, skills, education, projects, experience, and socials that can connect with the applicant. It contains a compilation of previous UI design projects as a proof of UI/ UX design skills development.

The project demonstrates my understanding of UI/UX design principles, encompassing clarity, consistency, and user-friendly navigation. Designed in Figma and intended for front-end development using HTML and CSS, it also serves as both professional portfolio and a practice project for improving my design and coding proficiency.

Objectives

To create a professional presentation for job applications.

To demonstrte proficiency in UI/UX dsign principles and front-end layout structuring.

To practice and enhance HTML and CSS implementation through an original design concept.

To serve as an evolving porfolio that shows growth in design and development capabilities.

Features

About me: A short biography including objectives that outlines the applicant’s background and career goals.

Skills: A list of technical and soft skills including the knowledge in specific tools and platforms.

Education: Academic background with awards and achievements.

Projects: A visual showcase of UI/UX works with detail descriptions.

Experience: Past internships or relevant work experiences

Socials: Links to professional and creative platforms for networking

PDF Resume Download and Email Button: Quick access for employers to download applicant’s resume and contact directly through Gmail.
