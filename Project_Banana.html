Banana Social App

Type: Academic Project — Mobile App UI Design

Subject Course: Project Management

Year: 2024

Role/Contributions: Main Ui Designer

Tool used: Figma

General Description

Banana is a Filipino-made social media mobile application that integrates popular features from platforms like Facebook Messenger, TikTok, Instagram, and an e-commerce system similar to Shopee. Named after the Philippines’ largest fruit export, the app symbolizes the country’s creativity and sweetness while reflecting Filipino identity. Most mainstream social media platforms are developed internationally, and existing local apps like Kumu primarily focus on live streaming. Banana aims to change that by offering an all-in-one platform that allows Filipinos to connect, express themselves, and support local businesses, showcasing that homegrown innovation can compete globally.

Objectives

Bringing Filipinos together through a locally developed social media app that embodies patriotism, diversity, and innovation.

Inspire Filipinos to believe that they can create platforms that compete internationally.

Support and promote local businesses through digital presence.

Target Users

All Flipinos around the world
